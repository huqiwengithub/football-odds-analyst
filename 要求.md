##### 欧赔数理运算数据流（Skill 步骤3 直接使用）

```
1. GET /v4/odds?fixtureId=X
2. 从响应中提取任意博彩公司（推荐 Pinnacle/Bet365）的 marketId=101
3. 读取三个 outcome 的 price 字段：
   - outcomeId=101, playerId=0 → price → 主胜赔率
   - outcomeId=102, playerId=0 → price → 平局赔率
   - outcomeId=103, playerId=0 → price → 客胜赔率
4. 代入公式：
   隐含总概率 = 1/主胜 + 1/平局 + 1/客胜
   返还率 = 1 ÷ 隐含总概率
   真实主胜概率 = (1/主胜) ÷ 隐含总概率 × 100%
   （同理计算平局、客胜真实概率）
5. Sharp vs Soft 对比：分别对 Pinnacle 和 Bet365 执行步骤 2-4，计算欧赔离散度
```

##### 亚盘匹配度运算数据流（Skill 步骤4 直接使用）

```
1. 从步骤3得到主胜真实概率 → 查表换算理论亚盘
2. 从 /v4/odds 响应中筛选 handicap ≠ 0 且 mainLine=true 的市场
3. 读取该市场两侧 outcome 的 price → 亚盘水位
4. 对比「理论亚盘」vs「实际亚盘 handicap 值」→ 判定欧亚匹配度
5. 若差异 ≥ 0.25 球 → 触发背离陷阱排查
```

##### 历史变盘解析数据流（Skill 步骤5-6 直接使用）

```
1. GET /v4/historical-odds?fixtureId=X&bookmakers=pinnacle,bet365
2. 对于 marketId=101 的每个 outcome：
   - 按 createdAt 排序得到完整价格时间线
   - 第一条记录 = 初盘（最早可观测的赔率定位）
   - 最后一条 active=true 的记录 = 当前临场盘
   - 中间所有记录 = 完整的变盘轨迹
3. 对于亚盘 market：
   - 跟踪 handicap 值是否发生变更（升盘/降盘）
   - 跟踪两侧 price 的变化（升水/降水）
   - 对照 Skill 知识库(5)「临场变盘真伪判定规则」
4. limit 字段辅助判断：
   - 初盘高 limit → 机构初始风险敞口大 → 态度真实
   - 临场 limit 骤降 → 机构收紧风险 → 可能触发异常
```

## 三、内置永久知识库（盘口全套研判规则）

### （1）赔率核心数理计算公式

所有分析强制带入运算：

```
隐含总概率 = 1/主胜赔率 + 1/平局赔率 + 1/客胜赔率
剔除抽水真实单项概率 = 单项赔率倒数 ÷ 隐含总概率
返还率 = 1 ÷ 隐含总概率
```

行业标准阈值：
| 联赛级别 | 正常返还率区间 |
|---------|-------------|
| 主流五大联赛（英超、西甲、德甲、意甲、法甲） | 90% - 95% |
| 次级联赛（英冠、德乙、荷甲、葡超等） | 87% - 90% |
| 小众联赛（北欧、东欧、南美等） | 85% - 88% |

**基础认知**：赔率 ≠ 球队胜率，仅为机构赔付定价，必须剥离抽水还原真实概率。

### （2）欧赔转亚盘标准换算对照表

| 欧赔主胜区间 | 合理理论亚盘 |
|------------|------------|
| 1.70 - 1.85 | 主让 0.5/0.75 球 |
| 1.85 - 2.00 | 主让 0.25/0.5 球 |
| 2.00 - 2.20 | 平手 / 主让 0.25 球 |
| 2.30 及以上 | 平手盘 或 客让格局 |

**换算说明**：
- 让球数精确到 0.25 球（即：平手、平/半、半球、半/一、一球等）
- 欧赔为客场球队优势时，对应换算为主队受让盘口
- 盘口对比时允许 ±0.25 球的浮动区间，超出则视为背离

### （3）六大欧亚盘背离陷阱形态

**每条数据必须交叉核对以下六类陷阱：**

| # | 形态 | 特征 | 风险提示 |
|---|------|------|---------|
| 1 | 欧赔深、亚盘浅 = 诱上盘 | 欧赔体现强队优势，亚盘开浅盘降低入手门槛 | 刻意造热热门，防平局、客胜爆冷 |
| 2 | 欧赔浅、亚盘深 = 阻上真实看好 | 纸面优势一般，亚盘强行开大深盘+高水 | 抬高上手门槛分流资金，正路打出概率更高 |
| 3 | 欧赔平赔持续下压、亚盘同步开深盘 | 机构用胜负盘造势掩盖平局 | 平局为隐藏结果 |
| 4 | 初盘欧亚完全匹配，临场欧赔降主胜、亚盘同步退盘升水 | 标准诱杀套路 | 假意利好主队，机构规避赔付 |
| 5 | 强队合理深盘搭配水位 > 1.05 超高水 | 机构不愿承担赔付 | 上盘难赢盘，易小胜走盘或爆冷 |
| 6 | 无基本面利好支撑，弱队方向无故持续降水 | 纯人为资金造热诱盘 | 数据参考价值极低 |

### （4）初盘四大定性铁律

初盘数据为机构原始态度，优先级高于临盘：

1. **初盘深开，全程无降盘、无降水**：机构初始定位强硬，无诱盘空间，正路参考价值高
2. **球队纸面实力占优，初盘刻意浅开**：机构主动示弱，防范热门资金涌入，重点防冷
3. **初盘水位 1.10 以上高位试探开盘**：机构不看好让球方，打出概率偏低
4. **合理实力盘口搭配初盘 0.80 以下超低水**：机构提前锁定赔付，真实看好对应方向

---

### （5）临场1小时变盘真伪判定规则

| 变盘类型 | 方向判断 | 解读 |
|---------|---------|------|
| 升盘 + 同步降水 | ✅ 真实看好 | 主动压低赔付，优先参考上盘 |
| 升盘 + 同步升水 | ❌ 诱盘高危 | 虚抬盘面制造强势假象，典型诱热门 |
| 降盘 + 同步降水 | ❌ 陷阱形态 | 降低投注门槛吸引散户造热 |
| 降盘 + 同步升水 | ❌ 彻底看衰 | 机构彻底看衰上盘，反向结果优先考量 |

**定性**：标准大热诱盘

### （10）十大通用避坑铁律

1. 低赔率不代表稳赢，超低赔率爆冷属于行业常态
2. 全网舆论一致热门的场次，机构大概率顺势造热，必须防范冷门
3. 无伤病、赛程变动等消息支撑的突发变盘，仅为资金平衡风控操作，无实质指向性
4. 小众次级联赛交易量低，盘口人为操控痕迹重，数据可信度大幅下降
5. 多串关会指数级降低命中概率，优先单场、2串低风险思路
6. 脱离基本面的单独盘口、赔率变化不具备分析意义
7. 初盘原始数据真实度高于临场动态盘，临盘仅用于判断资金流向
8. 赛事过热必有诱盘，非热门冷门方向更容易存在价值空间
9. 水位长期异常浮动的场次，优先防范冷门打出
10. 机构调整赔率盘口只有两大目的：平衡两边投注资金、诱导散户资金流向

### （11）数据概率方向预判 —— 加权概率合成模型

> **定位**：基于六维评分 + 欧赔真实概率 + 初盘定性 + 背离信号，合成**数据概率方向**。非赛果预测，是概率推演。

#### 基础概率

直接从步骤 3 欧赔数理运算获取：
```
基础主胜 = 剔除抽水后真实主胜概率（如 47.27%）
基础平局 = 剔除抽水后真实平局概率（如 28.50%）
基础客胜 = 剔除抽水后真实客胜概率（如 24.23%）
```

#### 修正因子（每项 ± 调整幅度）

| 因子 | 触发条件 | 修正幅度 | 方向 |
|------|---------|:--:|------|
| **欧亚背离** | 命中背离陷阱（六大任一） | ±10% | 陷阱#2(阻上)→主胜+10%；陷阱#1(诱上)→主胜-10% |
| **初盘态度** | 命中初盘铁律 | ±8% | 铁律#1(深开不降)→主胜+8%；铁律#2(刻意浅开)→主胜-8% |
| **临场变盘** | 升盘+降水 / 降盘+升水 | ±7% | 升盘降水→主胜+7%；降盘升水→主胜-7% |
| **基本面一致性** | 赔率走势与基本面方向是否一致 | ±5% | 一致→同向+5%；严重冲突→反向+5%（基本面为锚） |
| **六维评分** | 总分 ≥4 → 置信度加成；总分 ≤2 → 置信度下降 | ±5% | ≥4→主推方向+5%；≤2→全方向置信度降级 |

#### 概率合成公式

```
预判主胜概率 = 基础主胜 + Σ(修正因子)
预判平局概率 = 基础平局
预判客胜概率 = 基础客胜 + Σ(修正因子反向)

注：修正因子只作用于胜负两端，平局保持基础概率
    修正后三向概率之和可能 ≠ 100%，归一化处理
```

#### 方向判定规则

| 修正后概率差 | 方向判定 | 置信度 |
|-----------|---------|:--:|
| 主胜-客胜 > 25% | 主队方向 | 🟢 高 |
| 主胜-客胜 15%-25% | 主队方向 | 🟡 中 |
| 主胜-客胜 5%-15% | 主队略优 | 🟠 低 |
| |主胜-客胜| < 5% | 无明显方向 / 平局概率高 | 🔴 极低 |
| 客胜-主胜 > 15% | 客队方向 | 🟡 中 |

#### 输出要求

- 必须展示计算过程（基础概率 → 各修正因子 → 修正后概率）
- 必须附带置信度区间（如：主队方向，置信度中，约 55-65%）
- 必须附带反向风险提示（如：若客胜修正因子失效，爆冷概率约 20-25%）
- 结尾重申：此为数据概率推演，非赛果预测

## 四、每场赛事强制标准化分析流程

两种数据源统一执行以下 **11 个步骤**，不可删减：

---

### 步骤 1：确认数据源并调取完整赛事数据

- 确认当前启用数据源模式
- 调取完整赛事数据（赔率、盘口、基本面）
- 标注数据来源与更新时间

**API模式额外操作**：校验API响应状态、数据延迟是否超阈值
**联网模式额外操作**：标注数据抓取页面来源URL、数据抓取时间戳、缺失项清单

---

### 步骤 2：基本面综合研判

梳理以下基本面因子，定下基础强弱基调：

1. 核心球员伤停情况（门将 > 中卫 > 后腰 > 主力前锋）
2. 赛事战意判断（争冠/保级/欧战/中游无欲/练兵）
3. 近 6-10 场主客场攻防战绩数据
4. 历史交锋克制关系（近 3-5 次直接交锋）
5. 客观外部干扰（一周双赛疲劳、长途客场、雨雪场地）

**输出格式**：以要点列出基本面核心结论，明确「基本面指向：利好主队/均衡/利好客队」

---

### 步骤 3：欧赔数理拆解

强制运算：
1. 列出当前主流机构主胜、平局、客胜赔率
2. 计算隐含总概率 = 1/主胜赔率 + 1/平局赔率 + 1/客胜赔率
3. 计算返还率 = 1 ÷ 隐含总概率
4. 计算剔除抽水后的真实单项概率：
   - 真实主胜概率 = (1/主胜赔率) ÷ 隐含总概率 × 100%
   - 真实平局概率 = (1/平局赔率) ÷ 隐含总概率 × 100%
   - 真实客胜概率 = (1/客胜赔率) ÷ 隐含总概率 × 100%

**输出格式**：以表格列出计算过程与结果，注明返还率是否处于行业正常区间

---

### 步骤 4：欧亚盘匹配度对比 + 背离陷阱排查

1. 根据欧赔主胜赔率，查表换算理论亚盘
2. 对比实际开出亚盘
3. 逐一排查六大欧亚背离陷阱形态（命中哪一条即标注）
4. 给出「欧亚匹配度」结论：**匹配 / 轻微偏离 / 严重背离**

**输出格式**：列出理论盘口 vs 实际盘口对比表 + 背离陷阱命中清单

---

### 步骤 5：初盘定位定性分析

1. 调取初盘历史数据（欧赔初盘 + 亚盘初盘 + 初盘水位）
2. 套用初盘四大定性铁律，逐条判断是否命中
3. 给出初盘态度定性：**机构初盘态度：看好主队/中立/看好客队**

**输出格式**：初盘数据表格 + 四大铁律逐条定性 + 综合结论

---

### 步骤 6：临场变盘 & 水位走势解读

1. 梳理从初盘到当前临场的完整变盘轨迹（API模式可获取全链路；联网模式标注「部分缺失」）
2. 套用临场变盘真伪判定规则，区分真实利好 vs 诱盘走势
3. 排查「初盘造势+临场反向反转」经典收割套路
4. 分析当前水位所处位置与走势力向

**输出格式**：变盘时间线 + 每次变盘定性 + 综合走势结论
