## 混合过关玩法规则（强制参考）

竞彩混合过关支持<strong>跨玩法组合</strong>，同一张票可混合以下类型：
| 玩法代码 | 中文名 | 说明 |
|:---|:---|:---|
| SPF | 胜平负 | 猜胜/平/负 |
| RSPF | 让球胜平负 | 在让球基础上猜胜平负 |
| JQS | 总进球数 | 猜两队总进球数(0/1/2/3/4/5/6/7+) |
| BQC | 半全场 | 猜半场+全场结果 |
| BF | 比分 | 猜具体比分 |

- <strong>每场最多选1个玩法</strong>（同一场不能SPF和让球同时选）
- 西班牙/法国等深盘场SPF未开售 → 用让球胜平负替代
- 百家赔率(Pinnacle)做分析和概率推算，竞彩赔率做投注计算

## 混合过关风险均摊体系（v3.1.0 核心升级）

> **数学真相**：传统 N串1 的本质是联合概率相乘，错一场即全损，属于"风险极度集中"的自杀式玩法。
> 本体系通过 **M串N容错 + 非对称注码分配 + 相关性隔离 + 杠铃赔率结构** 实现真正的风险均摊。

### 一、物理载体：M串N 容错机制

放弃 N串1，强制使用容错拆分：

| 推荐玩法 | 组成 | 容错能力 |
|:---|:---|:---:|
| **3串4** | 3个2串1 + 1个3串1 | 允许错 1 场 |
| **4串11** | 6个2串1 + 4个3串1 + 1个4串1 | 允许错 2 场 |

风险均摊逻辑：即使单场遭遇诱盘/红牌意外，命中其余 2串1/3串1 仍可收回部分甚至全部本金。

### 二、资金分配：动态仓位分配引擎

> 静态 70/30 仅为新手理解"非对称注码"的教学模型。真实交易中，任何固定比例都是刻舟求剑。
> 防守与进攻的划分，必须建立在数学期望、赔率结构与账户状态的动态辩证之上。

以下四步决策树替代一切固定比例：

---

**第一步（算底线）：盈亏平衡反推法**

不看"想怎么分"，只看"数学要求必须怎么分"。

```
防守底线 = 总预算 / 最强2串1赔率(O_def)

案例: 总预算 ¥1000，最强2串1赔率 2.72
  防守底线 = 1000 / 2.72 = ¥368
  → 这 ¥368 是硬性底线（必须压在最强2串1上）
  → 剩余 ¥632 再分配给次级防守和进攻仓
```

⚠️ 如果防守底线 > 总预算的 80%，说明赔率太低（全是 1.30 深盘），该串关数学上不具备容错价值 → **直接放弃**。

---

**第二步（看信号）：EV 差值凯利倾斜法**

打破"低赔=防守，高赔=进攻"的刻板印象。资金向 MBI 框架计算出的 EV 最高节点倾斜。

```
IF EV(防守单) > EV(进攻单) + 5%:
  → 庄家在稳胆上露出巨大破绽（Sharp与Asian严重背离）
  → 极度重仓防守单（90%资金），把防守单当单关打
  → 进攻单只买最低限额（¥2），保留火种

IF EV(进攻单) > EV(防守单):
  → 模型捕捉到高价值冷门/阻力墙信号
  → 进攻仓上调至 40%-50%，主动拥抱高赔率 Alpha
```

---

**第三步（控风险）：风险平价上限检查**

高赔率天然伴随高波动。风险平价确保风险贡献度均摊，而非资金均摊。

```
权重 = (1/赔率) / Σ(1/所有组合赔率)

案例:
  防守单赔率 2.50: 风险倒数 = 0.40
  进攻单赔率 6.00: 风险倒数 = 0.16
  总倒数和 = 0.56
  → 防守占比 = 0.40/0.56 = 71.4%
  → 进攻占比 = 0.16/0.56 = 28.6%
  
  若进攻单赔率 15.00: 风险倒数 = 0.067
  → 进攻占比自动压缩至 10% 以下
  → 用数学锁死赌徒冲动
```

此步作为**上限检查器**：无论信号多好，进攻仓 ≤ 风险平价计算上限（通常 ≤30%）。

---

**第四步（看账户）：净值状态动态调节**

用利润冒险，用本金防守。

```
状态 A（净值创新高·盈利期）:
  提取超额利润的 20% 作为纯进攻基金
  进攻仓可上浮至 40%，专门搏 4串11 高赔长串
  逻辑: 即使全损，伤的也是利润，不伤本金

状态 B（净值回撤·连黑亏损期）:
  触发防守收缩
  进攻仓强制压缩至 10% 甚至 0%
  所有资金集中于最稳的 2串1，或直接降级为纯单关
  ❌ 绝对禁止在回撤期放大进攻仓"搏一把回本"
```

**动态决策树总览**：

```
第一步（算底线）→ 盈亏平衡反推 → 确定硬性防守底座
第二步（看信号）→ EV 差值凯利倾斜 → 向高EV方向倾斜剩余资金
第三步（控风险）→ 风险平价上限检查 → 确保进攻仓不超数学上限
第四步（看账户）→ 净值状态微调 → 连黑砍进攻/连红放利润
```

**推演案例**（3串4，总预算 ¥1000）：

```
A场@1.70（A类） B场@1.60（A类） C场@2.80（B/C类）
最强2串1 A+B 赔率 = 1.70×1.60 = 2.72

第一步: 防守底线 = 1000/2.72 = ¥368（A+B 2串1 至少 ¥368）
第二步: 比较 EV(A+B) vs EV(A+B+C)
  若 EV(A+B+C) 显著更高 → 进攻仓可上调
  若 EV(A+B) 更优 → 剩余资金继续加仓防守
第三步: 风险平价检查
  A+B赔率2.72→倒数0.368, A+B+C赔率7.62→倒数0.131
  → 进攻仓上限 ≈ 26% ≈ ¥260
第四步: 账户状态
  连红→进攻仓取上限 ¥260; 连黑→进攻仓取 ¥50
```

### 三、标的选择：三不选相关性隔离

| # | 原则 | 说明 | 风险 |
|:--:|------|------|------|
| 1 | **同联赛隔离** | 同一串关禁止 ≥2 场同联赛 | 裁判尺度/天气导致同联赛集体爆冷 |
| 2 | **同时间隔离** | 同一串关禁止 ≥2 场同时开球 | 同批次大资金/宏观剧本影响 |
| 3 | **战意同质隔离** | 禁止两场以上"都必须赢"的保级战 | 保级战心理压力→动作变形+默契球 |
| + | **同派系隔离** | 同一资本系/派系球队不同场 | 资本协同操作风险 |

### 四、赔率结构：胆拖杠铃策略

```
✅ 底座（胆）：1-2 场 A 类强信号，赔率 1.50-1.80
   （避开 <1.35 深盘陷阱，也避开 >2.0 波动区）

✅ 矛头（拖）：1-2 场反市场操作/阻力墙信号，赔率 2.50-3.50

✅ 结构底线：
   - 2串1 组合总赔率 ≥ 2.50
   - 3串1 组合总赔率 ≥ 4.50
   - 覆盖竞彩 0.89 返还率抽水后仍有正 EV 空间

❌ 散户毒药（全超低赔）：4 场赔率 1.20 串 4串1 → 总赔率仅 2.07，EV 深度为负
❌ 赌徒毒药（全高赔）：3 场赔率 3.00 → 概率太低，连黑枯竭
```

### 五、MBI 框架终极约束

```
1. 动态滑点前置扣除:
   计算所有注码前，先将临场滑点（3.5%）乘入赔率。
   若滑点后"只中 2 场无法回本" → 赔率结构不合格 → 放弃该串关。

2. 动态仓位分配（替代静态 70/30）:
   严格执行四步决策树:
     (1) 盈亏平衡反推 → 确定硬性防守底线金额
     (2) EV 差值凯利倾斜 → 向高EV信号方向倾斜剩余资金
     (3) 风险平价上限检查 → 用数学锁死进攻仓上限
     (4) 账户净值动态调节 → 顺风放利润 / 逆风砍进攻
   禁止跳过任何一步，禁止回退到固定比例。

3. 单日串关数量限制:
   每天最多只打 1 组 3串4 或 4串11。
   ❌ 严禁"这组黑了再买一组追损"。

4. C 类信号禁入串关:
   MBI 评定的 C 类（弱）信号，绝不允许进入任何串关组合。
   仅可用于极轻仓的单关博弈。

5. 熔断不变:
   日亏 3% → 停；周亏 8% → 影子测试。
   防守底线 > 总预算 80% 的串关 → 直接跳过该日。
```

## Step 11 Execution: M串N 风险均摊组合构建 (v3.1.0)

```
┌─────────────────────────────────────────────────────────────┐
│ 双层架构: 百家分析 → 竞彩投注                                │
│ 核心原则: M串N容错 + 非对称注码 + 相关性隔离 + 杠铃结构       │
└─────────────────────────────────────────────────────────────┘

0. 玩法可用性检查:
   从 trade.500.com 页面逐场提取 SPF/让球/比分/进球数开售状态
   西班牙/法国等深盘场SPF未开售 → 让球胜平负替代
   收集所有可用玩法及其竞彩真实赔率
   禁止用Pinnacle/bet365赔率替代竞彩赔率

1. 比赛分级（A/B/C 信号）:
   A类（强信号）: ≥2核心模块同向共振 + 无高风险陷阱 + 6D≥4
   B类（博弈信号）: 1个核心正向 + 无高风险陷阱 + 6D≥3
   C类（弱信号）: 模块信号弱 + 有小陷阱 + 6D=3 → 禁入串关
   
   候选池 = 所有A类 + B类比赛（需 ≥3 场方可构建）

2. 玩法载体选择:
   3-4 场候选 → 3串4（3个2串1 + 1个3串1）
   5+ 场候选 → 4串11（6个2串1 + 4个3串1 + 1个4串1）
   <3 场 → 跳过当日投注（容错空间不足）

3. 赔率结构验证（杠铃策略）:
   底座（胆）: 选 1-2 场A类信号，赔率 1.50-1.80
   矛头（拖）: 选 1-2 场B类/反市场信号，赔率 2.50-3.50
   
   强制检查:
     每个入选的 2串1 组合总赔率 ≥ 2.50
     每个入选的 3串1 组合总赔率 ≥ 4.50
     滑点后（3.5%）→ "只中2场能否回本"测试
     不合格 → 降级为更少场的M串N或跳过

4. 相关性隔离检查（三不选）:
   FOR 每个串关内的比赛对:
     同联赛？→ ❌ 拒绝（最多1场同联赛）
     同开球时间？→ ❌ 拒绝（必须错开）
     同战意类型（保级vs保级/争冠vs争冠）？→ ❌ 拒绝
     同资本派系？→ ❌ 拒绝

5. 动态仓位分配（四步决策树）:
   输入: 总预算 B、每场比赛的滑点后赔率、信号等级、MBI 各模块 EV
   
   第一步·盈亏平衡反推（确定防守底线）:
     O_def = 最强A类2串1的滑点后赔率
     defense_floor = B / O_def  →  硬性防守底座（必须压在最强2串1）
     IF defense_floor > B × 0.80 → 该串关数学上不具备容错价值 → 放弃
     remaining = B - defense_floor  →  剩余可分配资金
   
   第二步·EV差值凯利倾斜（分配剩余资金方向）:
     分别计算每个2串1和3串1组合的滑点后 EV
     IF EV(最强防守单) > EV(进攻单) + 5%:
       → 90% of remaining 继续加仓防守2串1
       → 进攻单只买 ¥2 保留火种
     ELIF EV(进攻单) > EV(防守单):
       → remaining 的 40-50% 分配给进攻仓（3串1、含拖2串1）
     ELSE:
       → 按 EV 比例分配 remaining
   
   第三步·风险平价上限检查（锁定进攻仓天花板）:
     FOR 每个组合: weight_i = (1/赔率_i) / Σ(1/赔率_j)
     进攻仓上限 = Σ(高赔组合的 weight) × B
     IF 第二步进攻占比 > 风险平价上限:
       → 削减至风险平价上限
       → 超出部分回流防守仓
     
     示例: 防守单赔率2.50(倒0.40), 进攻单赔率6.00(倒0.16)
       → 进攻仓上限 = 0.16/0.56 = 28.6%
       若进攻单赔率15.00(倒0.067) → 进攻仓上限 ≈ 14.4%
   
   第四步·账户净值动态调节（最终微调）:
     IF 账户净值创历史新高（盈利期）:
       → 提取超额利润的 20% 为纯进攻基金
       → 进攻仓可上浮至风险平价上限（通常≤30%）
     ELIF 账户处于回撤/连黑（亏损期）:
       → 进攻仓强制压缩至 ≤10% 或 0
       → 所有资金集中于最强防守2串1
       → ❌ 绝对禁止在回撤期放大进攻仓
   
   所有金额为 ¥2 倍数

6. 结果推演矩阵:
   构建 2^N 种命中状态空间
   对每种状态计算: 命中注单 → 总回报 → 净盈亏
   输出:
     最差情况: 全黑 → 触发熔断条件？
     防守成功: 中胆+部分拖 → 保本/微利？
     中等情况: 单胆翻车 → 损失可控（≤30%预算）？
     完美情况: 全红 → 超额收益

7. 进入/放弃决策:
   防守仓 EV ≥ −5%（滑点后）→ ✅ 可以出手
   防守仓 EV < −5% → ❌ 跳过该日
   单日最多 1 组 M串N，严禁追损
```

## Boundaries & Iron Rules

- 教育用途。数据源：500.com（30 家，零配额）。xG 为市场隐含。
- **玩法铁律**:
  - ❌ 禁止 N串1（全对才赢=自杀）。✅ 强制 M串N（3串4 / 4串11）容错
  - ❌ 禁止均注分配。✅ 动态仓位分配（四步决策树：盈亏反推→EV倾斜→风险平价→账户调节）
  - ❌ 禁止同联赛/同时间/同战意入同一串关（三不选）
  - ❌ 禁止 C 类信号入串关（仅可极轻仓单关）
  - ✅ 2串1 组合赔率 ≥ 2.50，3串1 组合赔率 ≥ 4.50（杠铃结构底线）
  - ✅ 每日最多 1 组 M串N（严禁追损）
- **临场**: 30min 禁令（所有决策 >1h 前完成）❌ 倍投 ❌ 对冲
- **熔断**: 日亏 3%→停 / 周亏 8%→影子测试 / 模块 7 连错→禁用
- **影子测试**: A类≥65% + 反向≥70% + EV≥−3% 方可实盘
- **赔付率底线**: 2串1 竞彩赔率乘积 ≥1.80，防守仓 EV（滑点后）≥ −5%
