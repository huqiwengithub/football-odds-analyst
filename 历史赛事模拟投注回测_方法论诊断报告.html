四、方法论问题诊断

🔴 P0 SKIP 不是"跳过"而是"反向指标"

SKIP 仓位命中率仅 43.8%（32场中仅14场正确），显著低于随机概率50%。
当系统判定"应该跳过"时，实际上有56.2%的概率应该押注——说明阈值设置过于保守。
影响：系统因错误跳过错过了 18场 正确判断。

🔴 P0 OCI1_RISE（赔率上升）和OCI4_NEUTRAL（离散度中性）是反向指标

OCI1_RISE：64场命中率仅46.9% — 当Pinnacle赔率上升时，实际走势往往相反。
OCI4_NEUTRAL：49场命中率仅42.9% — 离散度中性不应是中性信号，而应是负面信号。
当前权重表将这两个模式按0加权处理，实际应该设负权重。

🟡 P1 STANDARD 仓位接近噪声（52.0%）

STANDARD 是最大仓位组（100场/56%），但命中率仅52.0%。在100场样本中，实际仅比随机多对2场。
问题：STANDARD 仓位是系统的主力输出仓位，但如果它只是"略好于抛硬币"，整个系统的信号质量就被拉低了。
分赛事看：2018世界杯STANDARD达到63.6%，但2022世界杯降至45.2%，2024欧洲杯降至47.2% — 极不稳定。

🟡 P1 偏离信号单调性被破坏

偏离信号本应 越多 → 越准确（冷门识别），但实际：
偏离0 = 64.1% → 偏离1 = 49.4% → 偏离2 = 48.6% → 完全倒置。
根因：OCI权重表中的偏离因子权重过大，把一个正常的变化信号（如正常的临场资金流入）误判为"偏离"。
结果：每标记一条偏离，命中率反而降低约15个百分点。

🟡 P1 冷门翻转命中率偏低（53.8%）

13场冷门翻转标记，仅7场正确。未达到冷门投注所需的最低60%阈值。
根因：当前偏离信号阈值≥2即可触发，但偏离2条的命中率仅48.6% — 说明这些"偏离"案件中超过一半不是真正的冷门前兆。
将阈值提升至≥3可过滤掉大部分假偏离信号（偏离3虽然样本仅1场但命中100%）。

🟡 P1 2024 欧洲杯与其他赛事差异显著

2024欧洲杯整体命中率50.9%，显著低于2018世界杯（59.4%）和2022世界杯（53.1%）。
最严重问题：欧洲杯SKIP命中率仅20%（5场中1场正确），STANDARD命中率仅47.2%（36场中17场正确）。
同一套参数在三个赛事上表现不一致，说明存在赛事感知偏差。

🟡 P1 每日投注金额无标准化分配

当前65个投注日中，单日投入从¥10到¥100波动，且与场次数不呈比例。
例如：3场比赛日投入¥90，但4场比赛日也可能仅投¥20。
问题：投注金额分配没有遵循标准化的"按仓位档次×固定本金"公式。
有些日子的CORE仓位被分配了少量资金，而有些日子的STANDARD仓位却占了大量资金。

五、方法论调整建议

建议1：仓位阈值重校准（优先级：高）

  1. CORE门槛从65%降至60% — 当前仅34场（19%）入选CORE。降至60%后预计覆盖约55-65场（30-36%），释放更多高质量信号。
  2. STANDARD拆分为两层：STANDARD_HIGH（55-65%信心分）和 STANDARD_LOW（50-55%信心分）。后者自动降为SKIP或仅限单关微仓（≤¥5），不入串关。
  3. SKIP改为"反向参考" — 命中率43.8%意味着反向有56.2%正确率。当系统触发SKIP时，提示"反向操作可选"。合并偏离信号≥2时标记为"冷门候选"。
  4. COLD_FLIP偏离阈值从≥2提升至≥3 — 偏离2条命中率仅48.6%，不足以触发冷门翻转。偏离3条（仅1场但100%）和更高应作为触发条件。

建议2：OCI权重表动态更新（优先级：高）

  1. OCI1_RISE（46.9%）→ 权重设为负值（例如 -0.15 logit）— 赔率上升不是信号，是反向信号。
  2. OCI4_NEUTRAL（42.9%）→ 权重设为负值（例如 -0.10 logit）— 离散度中性=市场意见分裂？实际是负面信号。
  3. OCI1_FLAT（64.1%）→ 权重提升（当前约+0.10→提升至+0.15 logit）— 赔率平稳且命中率高，应给予更高权重。
  4. OCI5_INDEPENDENT（63.6%）→ 权重提升（当前约+0.08→提升至+0.12 logit）— 独立性验证是强信号。
  5. 偏离因子权重大幅下调 — 偏离信号加1条→信心降低约15pp，这个惩罚太大了。应下调至每偏离1条降5pp。

建议3：偏离信号重构（优先级：高）

  1. 偏离信号5条中有2条是"假阳性" — 信号2（量价背离）和信号5（OU线下调）在正常比赛中也会经常触发。
  2. 引入"偏离质量"而非"偏离数量" — 当前用简单计数，但不同类型的偏离信号预测力不同。应加权：信号1（Pinnacle升水）权重2x，信号3（市场分歧）权重1.5x，信号2和5权重0.5x。
  3. 偏离≥1 → 自动触发冷门模式 — 当前偏离1条时，正确判断方向但被错误否决了（命中率49.4%）。偏离1条不应改变方向判断，仅把仓位从CORE降至STANDARD。

建议4：分赛事参数校准（优先级：中）

  1. 2024欧洲杯STANDARD命中率仅47.2% — 建议在欧洲杯模式下，STANDARD仓位自动降半仓（从50%→25%仓位乘数）。
  2. 2024欧洲杯SKIP命中率20% — 在欧洲杯模式下，SKIP标记应直接解释为"反向操作建议"（有80%概率反向正确）。
  3. 长期来看需要在OCI模式库中加入"赛事类型标签"，按赛事类型分组查权重表，而非一刀切。

建议5：每日投注金额标准化（优先级：中）

  1. 统一标准：每日最多 ¥100 总投入 = ¥80 串关 + ¥20 单关辅助。
  2. CORE仓位分配：每次出现CORE标记 → 全仓 ¥40-50 入串关核心腿。单日最多1个CORE入串关。
  3. STANDARD仓位分配：每次出现STANDARD → 半仓 ¥20-25 入容错腿。
  4. 冷门翻转分配：仅单关 ¥15-20，不入串关。
  5. SKIP标记：不投。但如果偏离≥3且反向概率>55%，作为冷门单关¥10。

建议6：置信度分档系统修复（优先级：低）

  1. 当前置信度值集中在50-60%区间（149/179场 = 83%），无法有效分档。
  2. 建议将置信度输出改为"原始值"+"百分位数排名"双输出。百分位数可更精细区分信号强弱。
  3. 加入置信度校准：输出置信度X%时，实际命中率应该在X±5%范围内。当前未验证。
