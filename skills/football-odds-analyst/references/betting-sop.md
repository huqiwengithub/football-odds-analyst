# 量化投注 SOP（五步标准作业程序）

> v3.5.0 | 将赌博思维升级为量化交易思维：
> 找正EV信号 → 设赔率底线 → 拆解独立标的 → 算非对称注码 → 依账户状态微调 → 滚动风控与独立复盘。

---

## 核心原则

```
M串N容错 + 动态仓位分配 + 相关性隔离 + 杠铃结构

推荐载体:
  3串4 = 3×2串1 + 1×3串1  (容错 1 场)
  4串11 = 6×2串1 + 4×3串1 + 1×4串1  (容错 2 场)

核心心法: "不投"和"少投"本身就是极具价值的交易决策。
         找不到正EV信号 → 空仓。赔率不达标 → 空仓。
         防守底线算不过去 → 空仓。全天空仓也是赢。
```

---

## 第一步：信号筛选与赔率硬过滤

找正EV信号，拒绝对"凑单"妥协。

### 1.1 玩法可用性检查

```
从 trade.500.com 逐场提取 SPF/让球/比分/进球数开售状态
深盘场 SPF 未开售 → 让球胜平负替代
收集所有竞彩真实赔率（禁止用 Pinnacle 替代）
```

### 1.2 比赛分级（A/B/C）

```
A类(强信号): ≥2 核心模块同向共振 + 无高风险陷阱 + 6D≥4
B类(博弈信号): 1 个核心正向 + 无高风险陷阱 + 6D≥3
C类(弱信号): ❌ 禁入串关（仅可极轻仓单关）
候选池 = A类 + B类（需 ≥3 场方可构建）
```

### 1.3 载体选择

- 3-4 场候选 → 3串4
- 5+ 场候选 → 4串11
- <3 场 → 跳过当日

### 1.4 相关性隔离（三不选）

| # | 原则 | 说明 |
|:--:|------|------|
| 1 | 同联赛隔离 | 同一串关禁止 ≥2 场同联赛 |
| 2 | 同时间隔离 | 同一串关禁止同时开球场次 |
| 3 | 战意同质隔离 | 禁止两场以上同类型生死战 |
| + | 同派系隔离 | 同一资本系/派系球队不入同一串关 |

### 1.5 赔率硬过滤（生死线）

| 类型 | 最低阈值 | 防御目标 | 未达标→动作 |
|:---:|:---:|:---|:---|
| **2串1** | **≥ 1.80** | 覆盖滑点+容错回本 | 放弃串关→降级单关/等待 |
| **3串1** | **≥ 4.50** | 抵消连乘滑点+进攻补偿 | 剔除弱信号→缩为2串1/空仓 |
| **4串11** | **核心2串1≥1.80** | 错2场仍有回血能力 | 核心不达标→全计划作废 |

> ⚠️ 滑点后复验：将 3.5% 滑点乘入后重新检查，不达标执行右侧动作。

## 第二步：逻辑拆分与独立定价

拒绝体彩终端机的"均注套餐"，夺回仓位控制权。

### 2.1 拒绝套餐

```
❌ 绝不在终端机上直接购买"3串4"或"4串11"套餐
   系统强制均注 → 违背非对称资金分配原则
```

### 2.2 物理拆分

```
3串4 → 拆为 3 个独立 2串1 + 1 个独立 3串1
4串11 → 拆为 6 个独立 2串1 + 4 个独立 3串1 + 1 个独立 4串1
每一注 = 完全独立的交易标的
```

### 2.3 独立定价

```
对每个拆分后的组合，分别计算:
  滑点后赔率 = 账面赔率 × (1 − 3.5%)^N  (N=串关场数)
  模型预估胜率 = Π(各场 MBI 综合概率)
  滑点后 EV = 滑点后赔率 × 胜率 − 1
```

## 第三步：EV 计算与非对称注码分配

让数学底线和信号质量共同决定每注买多少钱。

### 3.1 剔除无效注单

```
计算每个组合的滑点后 EV
若某组合 EV < −5% → 直接砍掉，不予分配资金
例: A+B+C(3串1) EV=−8% → 删掉，仅保留 3 个 2串1
```

### 3.2 锁定防守底线（盈亏平衡反推）

```
假设总预算 ¥1000，3串4 拆出 A+B、A+C、B+C（2串1）+ A+B+C（3串1）

O_def = 最强 2串1 滑点后赔率（假设 A+B=2.80）
defense_floor = 1000 / 2.80 ≈ ¥358

→ A+B 硬性分配 ¥360（确保错1场时 360×2.80=¥1008 覆盖本金）

⚠️ IF defense_floor > ¥800 → 赔率太低，直接放弃
remaining = 1000 − 360 = ¥640
```

### 3.3 按 EV 权重分配剩余资金

```
FOR 剩余有效组合，按各自 EV 占比分配 remaining
最终结果示例: A+B ¥360 / A+C ¥400 / B+C ¥240（极度非对称！）
```

### 3.4 风险平价上限检查

```
用赔率倒数加权法锁死进攻仓天花板 → 详见 KB-13.5b.3
所有金额 ¥2 倍数
```

## 第四步：账户状态微调

下单前，根据账户净值动态调节攻守比例。

### 4.1 顺风局（净值创新高·盈利期）

```
→ 提取超额利润 20% 为纯进攻基金
→ 进攻仓（3串1/4串1）可上浮至风险平价上限（≤30%）
→ 逻辑: 全损也只伤利润，不伤本金
```

### 4.2 逆风局（净值回撤·连黑亏损期）

```
→ 触发防守收缩
→ 强制砍掉所有 3串1 进攻仓
→ 全部预算集中在最强 2串1 防守单
→ 极端时可降级为纯单关
→ ❌ 绝对禁止在回撤期放大进攻"搏回本"
```

## 第五步：滚动执行与独立复盘

利用时间差动态风控，验证模型有效性。

### 5.1 滚动风控（分时段比赛的独特优势）

```
若比赛分时段进行（如 18:00 / 20:00 / 22:00）:
  18点 A 场黑了 → 立刻取消或缩减后续含 A 场的未开赛注单（如 A+C）
  18点 A 场红了 → 后续正常执行，不追加

逻辑: 用已完成的赛果做实时止损，这是单关不具备的结构性优势
```

## 附录：风控执行细则（从 KB-13.3/13.4/13.5 移入）

### 熔断规则

```
单日熔断:  日亏≥3%本金 → 全停 → 复盘
周度熔断:  周亏≥8%本金 → 停实盘 → 影子测试(1周胜率≥65%→恢复-半仓)
模块熔断:  单模块7连错 → 禁用 → 查根因 → 修复后恢复
```

### Risk Day Index — 哪天该空仓

```
一级空仓(全日停):
  ① ≥3场触发Trap#16 或 ≥4场触发Trap#20/#21
  ② 野鸡占比≥70% + DRI均值>50
  ③ 连续2天亏损到达日熔断×2

二级降仓(仓位×0.5, 仅2串1):
  ① 必发异动场次≥30%  ② 转会窗/国家队名单前24h
  ③ 联赛末3轮保级战/杯赛次回合首回合差≥3球
```

### 铁律红线

```
❌ 临场30分钟禁令: 所有决定在开赛前>1h必须完成
❌ 禁止倍投: 严格按Kelly, 单场硬上限5%
❌ 禁止无意义对冲: 竞彩双边同时下注=双倍抽水
❌ 串关几何平均≥1.50: 3腿全部<1.35→不准
❌ 竞彩EV≥−5%: 滑点+返奖率调整后不达标→不准
```

## 附录：M串N执行细则（从 KB-13.5b 移入）

### 动态仓位分配四步决策树

```
Step 1（算底线）→ Break-Even Reverse:
  防守底线 = 总预算(B) / 最强2串1滑点后赔率(O_def)
  IF 防守底线 > B×80% → 放弃该串关

Step 2（看信号）→ EV Differential Kelly:
  IF EV(defense) > EV(attack)+5% → 90%加仓防守
  ELIF EV(attack) > EV(defense) → 40-50%转进攻
  ELSE → 按EV比例分配

Step 3（控风险）→ Risk Parity Cap:
  组合权重 = (1/odds_i) / Σ(1/odds_j)
  进攻仓上限 = Σ(高赔倒数)/Σ(总倒数)×B

Step 4（看账户）→ Account Equity:
  顺风: 提超额利润20%做进攻基金, 上限≤30%
  逆风: 进攻归零, 集中最强2串1防守
```

## 附录：信号分级与竞彩 EV 执行（从 knowledge-base.md KB-13.2/13.8 移入）

### 信号分级 (A/B/C)

```
| Tier | Criteria | Kelly Coeff | Single-Match Cap | Parlay Admission |
|:---:|------|:---:|:---:|:---|
| A (Strong) | ≥2核心模块同向共振+无高危陷阱+6D≥4 | 0.25× | 5% | 可为核心胆 |
| B (Medium) | 1核心正向+无高危陷阱+6D≥3 | 0.15× | 3% | 仅配A类 |
| C (Weak) | 弱模块正向+有轻陷阱+6D=3 | 0.08× | 1.5% | 不入串关, 仅轻仓单关 |

核心模块 = Lead-Lag / Water Flow / Exchange VWAP
高危陷阱 = #13,#16,#19,#20,#21 (≥1个触发=高危)
```

### EV 执行分级

```
EV > +0.05:       正期望，正常投
−0.05 ≤ EV ≤ +0.05: 边界，仅限小仓（≤1% 本金）
EV < −0.05:       负期望，标注 ⚠️ 警告，仓位 × 0.50
EV < −0.15:       严重负期望，🔴 禁止入选串关核心

策略含义:
  单关 EV 阈值: EV ≥ 0 即正期望
  串关 EV 阈值: 需 ≥ +0.05 才能覆盖返奖率差
  Step 11 串关构建前: 逐条腿计算竞彩 EV，任何 EV<−0.05 不得入选
```

## 附录：仓位分配规则（从 knowledge-base.md KB-18 移入）

### 三档仓位（v3.5.3 重构：模式驱动核心仓位）

```
仓位判定分两条路: 条件匹配优先, 信心分数后备。依据: 179场历史回测+2026世界杯44场验证。

  🔥 核心仓位 (全仓, 入串关核心) — 满足以下任意一条即可:

  路径A(模式匹配): 机构意见收敛 + 热门赔率未升 + 亚盘同步
    条件: OCI4=CONVERGE(离散收敛) AND OCI1≠RISE(热门不升水) AND OCI5=SYNC(亚盘同步)
    ─ 2026世界杯验证: 5场触发, 5/5=100%正确
    ─ 盘口含义: 30家机构意见趋于一致, Pinnacle不看好冷门, 亚盘同步确认

  路径B(信心分数): 信心分 >= 0.08 且 OCI中 >= 3个正向 且 OCI4≠DIVERGE
    ─ 给路径A未覆盖但多个指标同向的场次
    ─ OCI4=DIVERGE(离散发散)禁止进入核心: 12场中6/6=50%, 纯噪声

  🚫 跳过 (不投)
    条件: 不满足核心仓位条件, 且 不满足冷门翻转条件
    ─ 找不到正EV沉默同样是胜利

  🔄 冷门翻转 (仅押单关)
    条件: 信心分 < 0 且 偏离信号 >= 3
    动作: 反转系统方向, 押冷门/平局单关
    ─ 偏离≥3确保翻转的触发条件更严, 减少伪信号
```

### 串关资格

```
  核心仓位: 可入串关核心
  跳过: 不入任何串关
  冷门翻转: 仅单关 (翻转信号不够强, 不能串)
```
