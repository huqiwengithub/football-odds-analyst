### 术语翻译（强制——给用户看的必须用大白话）

| 内部术语 | 大白话输出 |
|:---|:---|
| SCS | 各家博彩公司是不是统一看法 |
| DRI | 公司和公司之间吵起来了没 |
| Lead-Lag | 谁先调价谁跟风调的 |
| Water Flow | 赔率水分怎么走的 |
| Exchange / 必发成交量 | 真正下注的钱流向了哪边 |
| Kelly | 庄家赔付压力大不大 |
| logit 校正 / Shin de-vig | 把庄家抽水去掉后算出的真实概率 |
| Poisson | 进球可能性计算 |
| 6D Score | 六个维度的综合评分 |
| MBI | 综合各家博彩公司的判断 |
| EV | 预期能赚多少 |
| AH / 让球盘 / 亚盘 | 强队让弱队几个球 |
| OU / 大小球 | 总进球数多还是少 |
| SPF / 1X2 | 主队赢 / 平局 / 客队赢 |
| Overround | 庄家抽了多少水 |
| P(全灭) | 血本无归的概率 |
| Resistance Wall | 有多少抛压在顶着 |
| Circuit Breaker | 亏到多少就该停手 |
| Trap | 盘口里的坑/陷阱 |
