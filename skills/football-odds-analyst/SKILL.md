## 13 步执行流程

> ⚠️ 每步必须输出结果。禁止"看起来简单就跳过"。每步标注要读取的 KB 章节。

### Step 1 — 加载数据
- 加载 `{date}_deep.json`
- 队名校验（KB-0 协议）
- 确认 6 页完整（ouzhi / yazhi / rangqiu / daxiao / shuju / touzhu）
- 输出：`N 场比赛数据就绪，[队名列表]`

### Step 1.5 — 反叙事筛查
- 新闻驱动？首秀/关键缺席？历史平局率 >27%？
- 参考 KB-5 平局基准线
- 输出：`[场次编号] 触发反叙事：[原因]` 或 `无反叙事信号`

### Step 2 — 基本面分析
- H2H / 近期状态 / 伤停 / 排名
- 权重 v2.0，参考 KB-5（基本面权重）和 KB-11（三档赛事参数）
- 输出：`[场次] 基本面倾向：[方向]，强度：[高/中/低]`

### Step 3a — MBI 多机构共识
- **必须先读 KB-10**
- 执行 SCS → DRI → Lead-Lag → WaterFlow → Exchange → Kelly
- 输出 MBI 面板（每模块判定 + 方向信号）

### Step 3b — 概率数学
- **必须先读 KB-1**
- Shin de-vig 去抽水 + 层级均值 overround 计算
- 输出：Pinnacle 真实概率、市场隐含概率、抽水比例

### Step 4 — 欧亚转换 + 陷阱
- **必须先读 KB-2 + KB-10 陷阱 #16-#21**
- 理论 AH vs 实际 AH 对比，15+6 条欧亚陷阱并行检测
- 输出：盘口偏差值 + 触发陷阱编号

### Step 5 — 开盘定位
- 开盘赔率 vs 公允价值，层级间对比
- 参考 KB-5
- 输出：`[场次] 开盘定位：[偏高/公允/偏低]，层级分歧：[是/否]`

### Step 6 — 临场变动
- 6h → 2h → 30min 水位趋势
- 回调验证 / AH 真假突破 / 初受盘差（KB-7 + KB-12）
- 输出：水位方向 + 突破真伪 + 初受盘差信号

### Step 7 — 6D 综合评分
- **必须先读 KB-4**
- 6 维度连续 0-6 分，维度去重叠
- 输出：每场 6 维得分 + 总分

### Step 8 — 陷阱全扫描
- **必须先读 KB-2 + KB-3 + KB-10**
- 49 条陷阱规则并行检测
- 输出：触发清单 + 判定（≥2→🔴 / ≥3→systemic）

### Step 9 — 方向总结
- 综合以上 → 每场方向判定
- 红 = 主胜 / 琥珀 = 平 / 绿 = 客胜，结论置顶
- 输出：`[场次] 方向：[胜/平/负]，把握：[高/中/低]`

### Step 10 — 概率精算
- **必须先读 KB-6 + KB-7**
- Logit 12 项校正 → 归一化 → Poisson → Top 3 比分
- 输出：每场概率分布 + 前 3 比分及概率

### Step 11 — 组合构建
- **详见下方 Step 11 完整章节**
- M串N 容错 + 四步决策树 + 三不选 + 杠铃结构
- 输出：过关方案 + 仓位分配 + 回报矩阵

### Step 12 — 报告生成
- **详见下方 Step 12 章节**
- HTML 输出：MBI 面板 + 6D 评分 + 比分 + 过关方案 + 免责声明
- 术语强制翻译（见 Output Format 术语表）

## Step 11: M串N 风险均摊组合构建 (v3.1.1)

> 详细公式和案例见 **KB-13**。以下为完整执行流程。
> 
> ⚠️ **开始前复查硬约束**：日熔断 ≤3%？周熔断 ≤8%？模块无禁用？距开球 >1h？

### 核心原则

```
M串N容错 + 动态仓位分配 + 相关性隔离 + 杠铃结构

推荐载体:
  3串4 = 3×2串1 + 1×3串1  (容错 1 场)
  4串11 = 6×2串1 + 4×3串1 + 1×4串1  (容错 2 场)
```

### 执行步骤

**0. 玩法可用性检查**

```
从 trade.500.com 逐场提取 SPF/让球/比分/进球数开售状态
深盘场 SPF 未开售 → 让球胜平负替代
收集所有竞彩真实赔率（禁止用 Pinnacle 替代）
```

**1. 比赛分级（A/B/C）**

```
A类(强信号): ≥2 核心模块同向共振 + 无高风险陷阱 + 6D≥4
B类(博弈信号): 1 个核心正向 + 无高风险陷阱 + 6D≥3
C类(弱信号): 信号弱/小陷阱/6D=3  →  ❌ 禁入串关（仅可极轻仓单关）

候选池 = A类 + B类（需 ≥3 场方可构建）
```

**2. 载体选择**

```
3-4 场候选 → 3串4
5+ 场候选 → 4串11
<3 场 → 跳过当日（容错空间不足）
```

**3. 赔率结构验证（杠铃策略）**

```
底座(胆): 1-2 场 A 类，赔率 1.50-1.80  (避开 <1.35 深盘和 >2.0 波动区)
矛头(拖): 1-2 场 B 类/反市场信号，赔率 2.50-3.50

强制底线:
  ✅ 2串1 组合总赔率 ≥ 2.50
  ✅ 3串1 组合总赔率 ≥ 4.50
  ⚠️ 滑点后(3.5%) → "只中 2 场能否回本" → 不合格则降级/放弃
```

**4. 三不选相关性隔离**

| # | 原则 | 风险 |
|:--:|------|------|
| 1 | 同联赛隔离（禁止 ≥2 场同联赛） | 裁判尺度/天气集体爆冷 |
| 2 | 同时间隔离（禁止 ≥2 场同时开球） | 同批次大资金/宏观剧本 |
| 3 | 战意同质隔离（禁止多场"必须赢"保级战） | 心理压力+默契球 |
| + | 同派系隔离（同一资本系不同场） | 资本协同操作 |

**5. 四步动态仓位决策树**（替代静态 70/30）

```
输入: 总预算 B、滑点后赔率、信号等级、各组合 EV

第一步·盈亏平衡反推（硬性防守底线）
  O_def = 最强 A 类 2串1 滑点后赔率
  defense_floor = B / O_def
  ⚠️ IF defense_floor > B × 80% → 赔率太低，不具备容错价值 → 放弃
  remaining = B - defense_floor

第二步·EV 差值凯利倾斜（剩余资金分配方向）
  IF EV(防守) > EV(进攻) + 5%:  → 90% remaining 加仓防守，进攻仅 ¥2
  IF EV(进攻) > EV(防守):       → 40-50% remaining 给进攻仓
  ELSE:                          → 按 EV 比例分配

第三步·风险平价上限检查（数学锁死进攻天花板）
  weight_i = (1/odds_i) / Σ(1/odds_j)
  进攻仓上限 = Σ(高赔组合 weight) × B
  IF 第二步进攻占比 > 上限 → 削减至上限，超出回流防守

第四步·账户净值动态调节（利润冒险/本金防守）
  盈利期: 提取超额利润 20% → 纯进攻基金，进攻可上浮至上限(≤30%)
  回撤期: 进攻强制 ≤10% 或 0%，全部集中最强防守 2串1
          ❌ 绝对禁止回撤期放大进攻"搏回本"
```

**6. 结果推演矩阵**

```
构建 2^N 命中状态空间，逐状态计算回报和净盈亏：
  最差(全黑 → 熔断?) / 防守成功(保本?) / 中等(单胆翻车损失 ≤30%?) / 完美(超额?)
```

**7. 进入/放弃决策**

```
防守仓 EV(滑点后) ≥ −5% → ✅ 出手
防守仓 EV(滑点后) < −5% → ❌ 跳过该日
单日最多 1 组 M串N，严禁追损
```

### MBI 框架终极约束

```
1. 滑点前置扣除: 计算所有注码前将 3.5% 滑点乘入赔率
2. 四步决策树: 禁止跳过任何一步，禁止回退到固定比例
3. 单日限制: 每天最多 1 组 M串N，严禁"黑了再追"
4. C类禁入串关: 仅可用于极轻仓单关博弈
5. 熔断不变: 日亏 3%→停 / 周亏 8%→影子测试
```

## Boundaries & Iron Rules

- 教育用途。数据源：500.com（30 家，零配额）。xG 为市场隐含。
- **玩法铁律**：
  - ❌ 禁止 N串1 → ✅ 强制 M串N（3串4 / 4串11）容错
  - ❌ 禁止均注分配 → ✅ 四步动态决策树（盈亏反推 → EV 倾斜 → 风险平价 → 账户调节）
  - ❌ 禁止同联赛/同时间/同战意入同一串关（三不选）
  - ❌ 禁止 C 类信号入串关（仅可极轻仓单关）
  - ✅ 2串1 组合赔率 ≥ 2.50，3串1 组合赔率 ≥ 4.50（杠铃结构底线）
  - ✅ 每日最多 1 组 M串N（严禁追损）
- **临场**：30min 禁令（所有决策 >1h 前完成）❌ 倍投 ❌ 对冲
- **熔断**：日亏 3% → 停 / 周亏 8% → 影子测试 / 模块 7 连错 → 禁用
- **影子测试**：A类 ≥ 65% + 反向 ≥ 70% + EV ≥ −3% 方可实盘
- **赔付率底线**：2串1 竞彩赔率乘积 ≥ 1.80，防守仓 EV（滑点后）≥ −5%
